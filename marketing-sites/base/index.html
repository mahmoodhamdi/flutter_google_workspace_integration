What's inside

Production-grade Flutter app with everything you'd build yourself in 200 hours, ready in 5 minutes.

📅 7 Workspace integrations

Calendar (full CRUD + recurrence + reminders), Drive (with progress callbacks), Sheets, Gmail (send-only), Contacts, Maps, Meet. All using SENSITIVE scopes only — no CASA assessment required.

🔐 Multi-account auth

Google Sign-In with workspace scopes, Firebase email/password fallback, biometric lock, secure token storage (Keychain/Keystore), automatic token refresh, scope expansion flow.

🏗️ Clean architecture

Domain / data / presentation layers per feature. Riverpod for state. freezed for immutability. dartz Either for typed errors. go_router for declarative navigation.

📦 Offline-first cache

Encrypted Hive boxes per feature with AES-256 (key in secure storage). Sync queue auto-flushes pending writes when connectivity returns.

🌍 AR + EN localization

Full Arabic + English support with RTL layouts. Material 3 theme system, dark mode, dynamic colors per-vertical.

✅ 170+ tests

Unit, widget, golden, and integration test scaffolding. Strict analyzer with 40+ lint rules. GitHub Actions for analyze, test, build matrix, release.

🎯 5 ready-to-ship verticals

Use as a starter kit, or build BizCalendar, DriveVault, SheetsOps, or MeetCompanion as their own products from the same codebase via Flutter flavors.

📡 Robust networking

Dio with auth interceptor (auto-refresh on 401), exponential-backoff retry on transient errors, typed error mapping for every status code.

📊 Full CI/CD

analyze.yml, test.yml, build.yml (Android+Web+iOS matrix per flavor), release.yml (auto-publish artifacts on v* tag).
